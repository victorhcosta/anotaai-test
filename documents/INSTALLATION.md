# Installation

## Prerequisites

- Node.js
- npm or yarn

## Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/victorhcosta/anotaai-test.git
   cd anotaai-test
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Access http://localhost:4200/ in your browser.
